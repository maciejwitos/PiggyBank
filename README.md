# PiggyBank

Is an app to manage you home budget with maximum customisation. Just create account and start using it. 
- add transactions
- categories
- accounts in various currencies
- create and plan you monthly budgets for each category


## Technology


```bash
python3
django 2.2.7
```
## Requirements
